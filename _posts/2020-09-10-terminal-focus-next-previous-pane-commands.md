Switch focus between split terminal instances using the keyboard.

Windows / Linux: <kbd>alt</kbd> + <kbd>left/right arrow</kbd>
MacOS: <kbd>cmd</kbd> + <kbd>option</kbd> + <kbd>left/right arrow</kbd>
