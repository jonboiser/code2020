Collapse all block comments in the current file. Use the `Unfold all` command to expand them again.

Windows + Linux: <kbd>ctrl</kbd><kbd>k</kbd> <kbd>ctrl</kbd><kbd>/</kbd>

Mac: <kbd>cmd</kbd><kbd>k</kbd> <kbd>cmd</kbd><kbd>/</kbd>
