Use `--goto` on the command line to open a file in VS Code at a specific line number. Example syntax:

```bash
$ code --goto FILE_PATH:LINE
```

Optionally also specify the column too:

```bash
$ code --goto FILE_PATH:LINE:COLUMN
```
