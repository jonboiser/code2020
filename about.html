About

Code 20/20 brings you tips and tricks for working productively in VS Code.

The site includes hundreds of short videos that coverer a wide range of topics, everything from multi-editing to customizing a theme. The videos are also aimed at a range of skill levels, from those just getting started with VS Code to experts who use VS Code everyday.

Creator

This project was created by me, Matt Bierner. I've been on the VS Code team since 2016, primary working on VS Code's JavaScript and TypeScript support but also working extensively on VS Code's extension APIs and also touch a number of other areas of the product.

I created Code 20/20 to share some of knowledge and tricks I use everyday while working in and on VS Code. VS Code is a very deep product and much of this depth is not immediately obvious. I want to help people discover more of the cool features that the VS Code team and community have implemented.

With Code 20/20, you can be sure the content was created by someone who generally knows what they are talking about. My goal is to teach you quickly and efficiently without the gimmicks and other BS that plague so many other coding resources.

Have a tip?

If you'd like to suggest a tip or topic for Code 2020, please submit a GitHub issue.

If the topic hasn't already been covered and seems like it would be interesting to a range of VS Code users, I'll try adding it to the site.

Have a question?

If you have a question about using VS Code or coding in general, ask on Stack Overflow.

This way your question can help all the others out there who may have a similar question.
